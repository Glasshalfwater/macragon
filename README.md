# Macragon

### Hexagonal macropad powered by a Waveshare 2040-zero with multiple keyboard layers, USB-C connectivity, RGB LEDs, and planned voice keyboard functionality.

![](/assets/images/fallout_zine_image.png)(/assets/fallout_zine_macropad.pdf)

It is said that laziness is the mother of invention, and in this case it is also the father. I mainly wanted to make this project because I wanted to make CADing in engineering class easier. I also would like to integrate a voice keyboard so I can finally slouch away from my desk while writing.
